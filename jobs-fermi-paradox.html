Fermi Paradox and Job Searching

Introduction

The "Fermi Paradox" is the question of why no alien life has been found despite the universe being so vast.
Similarly, the world economy contains so many businesses and industries, but jobs can be scarce.
Why is finding a job so hard if there are so many job openings?

A way to answer the Fermi Paradox is provided by the "Drake Equation":
N = (R) x (fp) x (ne) x (fl) x (fi) x (fc) x (L)

A detailed explanation of the Drake Equation would be too long here, but for more information check out:
https://en.wikipedia.org/wiki/Drake_equation

This approach can be adapted for solving the job search problem.

Job Search Process

There are multiple phases in a typical hiring process which need to be successfully passed by the applicant:

Application (Resume and Cover Letter)

Preparing the application is the initial phase in which the job seeker is fully in control.
No one can reject the application at this phase except the applicant themself.

Keep in mind some basics:
The resume is 100% about the candidate.
Tthe job requirements are 100% about the employer.
The cover letter is the bridge connecting the resume and job requirements.
